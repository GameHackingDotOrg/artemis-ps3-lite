# gui
Frontend of Artemis PS3 Lite.


# Dependencies
	        PSL1GHT SDK (https://github.com/HACKERCHANNEL/PSL1GHT)
	        Tiny3D (https://github.com/wargio/tiny3D)
	        Mini2D (https://github.com/Dnawrkshp/mini2d)


# Build
~~~~
make
~~~~

# Run
~~~~
make run
~~~~

# Package
~~~~
make pkg
~~~~
